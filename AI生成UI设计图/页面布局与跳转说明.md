# 方寸智纳页面布局与跳转说明

本文基于 `页面元素图（部分）.png`、`闲置（首页）.png`、`功能清单.png` 整理，目标是先确定各页面的布局骨架与页面跳转关系，便于下一步统一产出最终 UI 页面图。

## 1. 整体信息架构

底部主导航建议固定为 5 个一级入口：

- 首页：综合入口，承接 AI 收纳、热门推荐、社区、后勤服务等总览内容。
- 收纳：AI 收纳规划主入口。
- 闲置：校园闲置循环市集主入口。
- 社区：打卡、帖子、互动、积分激励入口。
- 我的：个人资料、订单、方案、发布、收藏、消息、设置入口。

全局通用组件建议：

- 顶部导航栏：返回、标题、搜索、消息、筛选、分享等按页面需要组合。
- 卡片系统：统一使用圆角卡片承载商品、方案、帖子、服务。
- 标签系统：用于分类、筛选、场景标签、成色标签、状态标签。
- 底部按钮区：用于发布、保存、确认预约、支付、分享等强操作。
- 底部导航栏：仅一级页面保留，二级详情页默认隐藏。

整体视觉方向建议沿用 `闲置（首页）.png` 的风格：

- 浅米白纸张背景，带轻微网格感。
- 主色以豆绿色为主，辅以浅黄、浅橙、浅紫做分类区分。
- 图标与插画偏手绘、校园感、生活化。
- 大卡片、大留白、圆角边框，营造轻松友好的校园产品气质。

## 2. 页面布局说明

说明：线框图中有 2 个标题较模糊的页面，本文按功能命名为“收纳方案详情页”和“定制需求填写页”。

### 2.1 首页模块

#### 1）首页

- 布局：顶部为 `logo + 搜索框 + 消息入口`；中部为横向功能快捷区，至少包含 `AI 助手`、`闲置集市`；下方为内容分区，包含 `社区`、`好物推荐与安全科普`、`后勤服务` 等入口卡片；底部保留五栏主导航。
- 跳转：搜索框进入搜索页；消息入口进入消息页；AI 助手进入 AI 收纳主页；好物推荐区进入好物推荐与安全科普页；后勤服务区进入后勤服务页；底部导航切换到收纳、闲置、社区、我的。

#### 2）搜索页

- 布局：顶部为 `返回 + 搜索输入框`；其下为 `历史搜索`；中部为 `系统推荐内容/猜你想看` 列表；底部为搜索结果区，可根据搜索词展示方案、商品、帖子、服务。
- 跳转：返回上一级；点击历史词重新触发搜索；点击结果项按内容类型进入收纳方案详情页、商品详情页、帖子详情页或后勤服务详情页。

#### 3）消息页

- 布局：顶部为 `返回 + 搜索 + 消息分类筛选`；中部是消息列表，按 `系统通知`、`评论回复`、`聊天会话` 等分类展示。
- 跳转：点击系统通知可进入订单/方案/活动相关页；点击会话进入聊天页；从首页、我的页、闲置首页右上角消息图标都可到达此页。

### 2.2 AI 收纳模块

#### 4）AI 收纳主页

- 布局：顶部为 `logo + 全局搜索 + 消息`；核心功能区分为两块：`热门收纳` 与 `历史方案`；中部突出 `安全性检查` 卡片；下方为大尺寸 `上传照片/视频` 入口；底部保留主导航，当前高亮“收纳/方案”。
- 跳转：点击热门收纳/历史方案进入热门收纳历史方案页；点击安全性检查进入安全性检查页；点击上传入口进入上传照片页；底部导航可切换首页、闲置、社区、我的。

#### 5）上传照片页

- 布局：顶部为 `返回 + 生成方案`；中部依次为 `照片/视频上传`、`关键词选择或输入`、`草图导入` 等表单区；底部固定生成按钮。
- 跳转：点击生成方案进入收纳方案详情页；返回 AI 收纳主页。

#### 6）收纳方案详情页

- 布局：顶部为返回和页标题；主体从上到下为 `方案一/方案二/方案三切换`、`图片/3D 预览区`、`文字说明`、`工具推荐`、`保存/分享/购买链接` 操作区。
- 跳转：返回上传照片页或历史方案页；点击分享唤起分享弹出框；点击购买链接进入商品详情页或商品列表页；保存后可沉淀到“我的方案”。

#### 7）热门收纳历史方案页

- 布局：顶部为 `返回 + 搜索`；中部提供 `精选栏位/筛选标签`；主体为历史方案列表。
- 跳转：点击任一历史方案进入收纳方案详情页；返回 AI 收纳主页。

#### 8）安全性检查页

- 布局：顶部为 `返回 + 检查`；表单区包含 `照片/视频上传`、`关键词选择`、`草图导入`。
- 跳转：点击检查进入检查结果页；返回 AI 收纳主页。

#### 9）检查结果页

- 布局：顶部返回；主体展示 `安全等级`、`整改建议`、`工具推荐`、`保存分享`。
- 跳转：可返回安全性检查页重新上传；可将推荐工具跳转到商品列表页或商品详情页；可分享检查结果。

### 2.3 好物推荐与安全科普模块

#### 10）好物推荐与安全科普页

- 布局：顶部为 `logo + 搜索 + 消息`；中上部为场景标签导航，如 `好物标签墙`、`场景服务`、`收纳打卡`；中部先展示 `热门商品推荐`；其下为 `场景分类按钮`，如宿舍、办公室、实验室、卫生间；底部为 `安全科普专栏`。
- 跳转：点击场景分类进入商品列表/安全科普页；点击商品卡进入商品详情页；点击安全科普文章进入安全科普详情页；点击收纳打卡进入收纳打卡页。

#### 11）商品列表/安全科普页

- 布局：顶部为 `返回 + 搜索`；中部是标签筛选；主体为商品列表或科普列表，按选择的场景或分类展示。
- 跳转：点击商品进入商品详情页；点击科普内容进入安全科普详情页；返回好物推荐与安全科普页。

#### 12）商品详情页

- 布局：顶部为 `返回 + 收藏 + 分享`；主体依次为 `商品主图/轮播`、`商品说明`、`购买评价区`；底部为 `类似好物分享`、`私聊/立即购买` 操作。
- 跳转：分享唤起分享弹出框；点击私聊进入聊天页；点击立即购买进入购买页；点击类似好物可回到商品列表页或进入其他商品详情页。

#### 13）安全科普详情页

- 布局：顶部为 `返回 + 收藏 + 分享`；主体以大图或插图开头，下面是正文说明；底部为评论区。
- 跳转：可分享；评论互动后沉淀到社区动态；返回商品列表/安全科普页。

### 2.4 社区与激励模块

#### 14）收纳打卡页

- 布局：顶部为 `logo + 搜索 + 消息`；中上部保留标签切换，如 `打卡标签墙`、`精品推荐`、`收纳打卡`；主体展示热门话题；中部为分类按钮，如 `宿舍`、`办公室`、`实验室`、`卫生间`；底部为发布入口。
- 跳转：点击分类可切换话题流；点击发布进入帖子发布页；点击打卡任务可进入任务中心/积分商城页；点击帖子卡进入帖子详情页。

#### 15）帖子发布页

- 布局：顶部为 `返回 + 发布`；中部依次为 `标题`、`内容`、`话题标签`、`图片上传`。
- 跳转：发布成功后进入帖子详情页，或回到收纳打卡页并插入新内容。

#### 16）帖子详情页

- 布局：顶部为 `返回 + 收藏 + 转发`；主体展示 `帖子内容`、`收纳对比图/视频`、`心得`、`话题`；底部为评论区和点赞按钮。
- 跳转：可评论、收藏、分享；返回收纳打卡页；分享时唤起分享弹出框。

#### 17）任务中心/积分商城页

- 布局：顶部为返回；中部显示当前 `积分`；下方为 `任务/商品列表`，可在任务列表与兑换商品列表之间切换。
- 跳转：点击任务可回到对应功能页完成任务，如帖子发布页、收纳打卡页；点击商品进入商品详情页或直接进入积分商城页。

#### 18）积分商城页

- 布局：顶部为 `logo + 搜索 + 消息`；中部为热门兑换商品；下方为分类栏，如 `书籍`、`生活用品`、`其他`；底部为发布或查看兑换明细等辅助入口。
- 跳转：点击分类进入商品列表页；点击兑换商品进入商品详情页；点击规则/兑换记录可进入订单或记录列表页；与任务中心互通。

### 2.5 后勤服务模块

#### 19）后勤服务页

- 布局：顶部为 `logo + 全局搜索 + 消息`；中上部为服务分类标签；中部展示 `上门收纳`、`后勤服务`、`收纳打卡` 等组合入口；下方为服务介绍和表单入口。
- 跳转：点击上门收纳进入上门收纳预约页；点击定制型服务进入定制需求填写页；点击推荐内容可跳回社区或好物模块。

#### 20）上门收纳预约页

- 布局：顶部为返回；中部是预约表单，包含 `服务信息选择`、`时间选择`、`地点`；底部为支付或提交预约按钮。
- 跳转：点击支付进入购买页；预约成功可回到我的主页中的订单页。

#### 21）定制需求填写页

- 布局：顶部返回；主体为 `效果类型选择`、`文字描述`、`图片上传`、`个人信息填写`；底部为确认按钮。
- 跳转：提交后可进入后勤服务页的服务单详情，或沉淀到我的订单列表。

### 2.6 闲置循环市集模块

#### 22）闲置首页

- 布局：参考 `闲置（首页）.png`。顶部为 `品牌 logo + 搜索框 + 消息`；首屏是校园感插画 Banner；其下为分类胶囊按钮，如 `书籍`、`数码`、`生活用品`、`宿舍好物`、`毕业季专区`；内容区采用双列瀑布流商品卡片；在商品流前插入一个醒目的 `发布闲置` 大卡片；底部保留主导航，当前高亮“闲置”。
- 跳转：点击搜索进入搜索页；点击消息进入消息页；点击分类进入商品列表页；点击发布闲置进入闲置发布页；点击商品卡进入商品详情页；点击毕业季专区可进入专区列表。

#### 23）闲置发布页

- 布局：顶部为 `返回 + 发布`；主体包含 `商品图上传`、`商品描述`、`类目标签`、`价格/交换条件`；底部为发布按钮。
- 跳转：发布成功后进入商品详情页，且同步出现在“我的发布”；返回闲置首页。

#### 24）商品列表页

- 布局：顶部为 `返回 + 搜索`；中部为 `筛选栏`；主体为商品列表。
- 跳转：点击任一商品进入商品详情页；返回闲置首页或积分商城页。

#### 25）聊天页

- 布局：顶部为 `返回`；主体为对话区；底部为输入框和发送按钮。
- 跳转：从商品详情页发起私聊进入；聊天完成后可回商品详情页或继续下单。

#### 26）购买页

- 布局：顶部返回；主体展示 `订单信息`、`商品`、`金额`、`支付方式`；底部为支付按钮。
- 跳转：支付完成后返回订单/记录页；支付前可返回商品详情页或预约页。

### 2.7 我的模块

#### 27）我的主页

- 布局：顶部左侧为品牌或头像，右侧为设置按钮；中部为用户卡片，展示 `头像、昵称、院校/身份、个性签名`；其下为 `VIP、积分、等级、订单` 快捷入口；再往下是 `我的方案`、`我的发布`、`我的收藏` 等列表入口；底部保留主导航。
- 跳转：设置按钮进入设置页；积分进入积分商城页；订单进入订单/我的方案/我的发布/我的收藏页；我的方案可进入收纳方案详情页；我的发布可进入商品详情页或帖子详情页。

#### 28）订单/我的方案/我的发布/我的收藏页

- 布局：顶部为 `返回 + 搜索`；中部用标签页切换 `订单`、`我的方案`、`我的发布`、`我的收藏`；主体为对应内容列表。
- 跳转：点击订单项进入购买页或订单详情；点击方案进入收纳方案详情页；点击发布内容进入商品详情页或帖子详情页；点击收藏内容按类型进入商品详情页、帖子详情页或方案详情页。

#### 29）设置页

- 布局：顶部返回；主体为纵向设置菜单，包含 `账号与安全`、`地址`、`通知设置`、`隐私设置`、`账号与客服`、`切换/退出登录`。
- 跳转：进入对应设置子页；退出登录回到登录页。

#### 30）分享弹出框

- 布局：居中弹窗，包含分享提示文案、分享渠道按钮、取消按钮。
- 跳转：从商品详情、帖子详情、方案详情、安全科普详情等页唤起；选择渠道后完成分享并回到原页。

## 3. 页面跳转主链路

### 3.1 主导航链路

- 首页 ↔ 收纳 ↔ 闲置 ↔ 社区 ↔ 我的

### 3.2 AI 收纳链路

- 首页/收纳入口 → AI 收纳主页
- AI 收纳主页 → 上传照片页 → 收纳方案详情页
- AI 收纳主页 → 热门收纳历史方案页 → 收纳方案详情页
- AI 收纳主页 → 安全性检查页 → 检查结果页
- 收纳方案详情页 → 分享弹出框
- 收纳方案详情页 → 商品详情页

### 3.3 好物推荐链路

- 首页 → 好物推荐与安全科普页
- 好物推荐与安全科普页 → 商品列表/安全科普页
- 商品列表/安全科普页 → 商品详情页
- 商品列表/安全科普页 → 安全科普详情页
- 商品详情页 → 聊天页 / 购买页 / 分享弹出框

### 3.4 社区链路

- 首页/社区导航 → 收纳打卡页
- 收纳打卡页 → 帖子发布页 → 帖子详情页
- 收纳打卡页 → 任务中心/积分商城页
- 帖子详情页 → 分享弹出框

### 3.5 后勤服务链路

- 首页 → 后勤服务页
- 后勤服务页 → 上门收纳预约页 → 购买页
- 后勤服务页 → 定制需求填写页
- 完成预约/提交需求 → 我的主页 → 订单页查看记录

### 3.6 闲置市集链路

- 闲置首页 → 商品列表页 → 商品详情页 → 聊天页 / 购买页
- 闲置首页 → 闲置发布页 → 商品详情页
- 闲置首页 → 搜索页
- 闲置首页 → 消息页

### 3.7 我的页面链路

- 我的主页 → 设置页
- 我的主页 → 积分商城页
- 我的主页 → 订单/我的方案/我的发布/我的收藏页
- 订单/我的方案/我的发布/我的收藏页 → 对应详情页

## 4. 下一步出图建议

下一步生成最终 UI 页面图时，建议先按以下优先级出图：

- 第一批：闲置首页、AI 收纳主页、我的主页、好物推荐与安全科普页、收纳打卡页。
- 第二批：搜索页、商品详情页、帖子详情页、上传照片页、收纳方案详情页。
- 第三批：安全性检查页、检查结果页、后勤服务页、预约页、积分商城页、设置页、消息页。

这样能先把 5 个一级入口和最关键的详情页视觉体系统一，再补充功能页。
